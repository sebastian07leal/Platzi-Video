# PlatziVideo

PlatziVideo es una aplicación construida en ReactJS como proyecto para la escuela de javaScript de [Platzi](https://platzi.com/)

## Instalación
```
npm install
```

## Ejecución
```
npm run start
```

## Compilar
```
npm run build
```
